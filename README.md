# Airbnb SF Listings

A responsive web app displaying the first 50 Airbnb listings in San Francisco, built with vanilla HTML, CSS and JavaScript.

## Live Demo
https://smit-velani.github.io/airbnb-sf-listings

## Features
- Loads 50 listings from JSON using JavaScript fetch and await (AJAX)
- Displays listing name, description, amenities, host name and photo, price and thumbnail
- Search by listing name or neighborhood
- Sort by price, rating or name
- Filter by room type
- Click any card to open a detailed modal
- Neighborhood quick filter tags
- About San Francisco section
- Fully responsive design

## Tech Stack
HTML | CSS | JavaScript | JSON | GitHub Pages
