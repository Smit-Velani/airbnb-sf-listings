# Airbnb SF Listings

A web app that displays the first 50 Airbnb listings in San Francisco loaded using JavaScript fetch and await (AJAX).

## Features
- Loads 50 listings from JSON using fetch/await
- Shows listing name, description, amenities, host name and photo, price, and thumbnail
- Search by name or neighborhood
- Sort by price, rating, or name
- Filter by room type
- Modal popup with full listing details
